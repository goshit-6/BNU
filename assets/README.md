# 素材替换目录

当前 demo 已使用提供的院徽与校园照片生成网页加载用的轻量副本，位于 `media/`；源图仍保留在项目上级目录中。背景照片在页面中叠加深绿色遮罩，以确保正文可读。

正式素材可按以下名称放置：

| 文件 | 用途 | 当前状态 |
| --- | --- | --- |
| `music.mp3` | 点击后播放的背景音乐 | 尚未放入，功能处于占位提示状态 |
| `media/emblem.png` | 学院院徽 | 已使用提供的院徽生成网页副本 |
| `media/hero-campus.jpg` | 首屏背景图 | 已使用校园照片并配合绿色遮罩 |
| `media/ginkgo.jpg` | 学院亮点区底图 | 已启用弱化展示 |
| `media/jingshi-hall.jpg` | 申请信息区底图 | 已启用弱化展示 |
| `media/motto-snow.jpg` | 费用与评优区底图 | 已启用弱化展示 |
| `media/official-wechat-qrcode.png` | 学院官方微信公众号二维码 | 已从学院官网资源接入末页信息卡 |
| `share-cover.jpg` | 微信分享缩略图/宣传封面 | 待发布链接与视觉定稿后制作 |

引入图片或音乐时，请同步进行手机网络加载速度与微信内打开测试。
